# Scripts

Quick start scripts for running Concierge components.

## Usage

### Start the server
```bash
./scripts/run-server.sh
```

### Start the UI
```bash
./scripts/run-ui.sh
```

### Run the client
```bash
./scripts/run-client.sh <api_base> <api_key>
```

Example:
```bash
./scripts/run-client.sh https://api.openai.com/v1 sk-...
```
